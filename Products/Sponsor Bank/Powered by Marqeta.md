# Marqeta in Europe Responsibility Matrix

> Describes the distribution of responsibilities between you and Marqeta for both PxM and MxM programs.

This page outlines the distribution of responsibilities between you and Marqeta when using either Powered By Marqeta (PxM) or Managed By Marqeta (MxM) for your card program. These responsibilities may change, depending on the contractual agreements between you and Marqeta.

| Responsibility                                                                                                                                                                          | Powered By | Managed By |
| --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------- | ---------- |
| Defining the Bank Identification Number (BIN) configuration with the card network/scheme                                                                                                | You        | Marqeta    |
| Approving and managing the card program with the card network/scheme                                                                                                                    | You        | Marqeta    |
| Approving and managing the card program with the issuing bank                                                                                                                           | You        | Marqeta    |
| Ensuring program compliance with regional regulations                                                                                                                                   | You        | Marqeta    |
| Ensuring program compliance with bank and card network mandates                                                                                                                         | You        | Marqeta    |
| Configuring the customer processing environment                                                                                                                                         | Marqeta    | Marqeta    |
| Configuring funding methodology                                                                                                                                                         | You        | Marqeta    |
| Configuring webhooks                                                                                                                                                                    | You        | Marqeta    |
| Issuing and fulfilling cards                                                                                                                                                            | You        | Marqeta    |
| Managing Know Your Customer (KYC)                                                                                                                                                       | You        | Marqeta    |
| Managing card inventory and fulfillment providers                                                                                                                                       | You        | Marqeta    |
| Managing tokenization with the card network                                                                                                                                             | You        | Marqeta    |
| Engaging with the digital wallet providers                                                                                                                                              | You        | Marqeta    |
| Reconciling with the card network                                                                                                                                                       | You        | Marqeta    |
| Monitoring transactions for fraud or money laundering                                                                                                                                   | You        | Marqeta    |
| Managing disputes                                                                                                                                                                       | You        | Marqeta    |
| Providing access to Marqeta APIs to ensure core functionality for integration                                                                                                           | Marqeta    | Marqeta    |
| Processing card network/scheme events such as authorization and clearing                                                                                                                | Marqeta    | Marqeta    |
| Processing real-time authorization requests from the card network and responding within the allotted time to secure successful transactions                                             | Marqeta    | Marqeta    |
| Processing network clearing events daily and distributing events to webhooks to complete ledger events and aid reconciliation                                                           | Marqeta    | Marqeta    |
| Providing Tier II technical support                                                                                                                                                     | Marqeta    | Marqeta    |
| Sending daily card network reporting files to the issuing bank                                                                                                                          | Marqeta    | Marqeta    |
| Sending crucial information regarding webhooks to keep your system synced with Marqeta                                                                                                  | Marqeta    | Marqeta    |
| Providing JIT Funding requests to process customer responses into network-specific approvals                                                                                            | Marqeta    | Marqeta    |
| Queuing and transmitting daily card orders to the card vendor                                                                                                                           | Marqeta    | Marqeta    |
| Creating and maintaining new or existing card products                                                                                                                                  | You        | Marqeta    |
| Providing Commando Mode solutions when the customer JIT gateway fails to improve authorization rates                                                                                    | Marqeta    | Marqeta    |
| Hosting and configuring a Three-Domain Secure (3DS) authentication platform to secure ecommerce transactions                                                                            | Marqeta    | Marqeta    |
| Hosting and configuring tokenization solutions for managing tokens and digital wallet payments                                                                                          | Marqeta    | Marqeta    |
| Providing access to the Marqeta Dashboard to complete core customer service functionalities for cardholders                                                                             | Marqeta    | Marqeta    |
| Providing fraud monitoring rules and solutions to enhance risk management                                                                                                               | Marqeta    | Marqeta    |
| Installing and maintaining Marqeta Bank Identification Number (BIN) configurations, as well as enabling BIN ranges to transact on the Marqeta platform as your program grows and scales | Marqeta    | Marqeta    |
